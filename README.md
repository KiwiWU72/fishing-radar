# 🎣 Jiādōng Wild Pond · Tactical Fishing Intelligence System (v4.0)
### 🌊 佳冬野生池 · 實戰釣魚戰術儀表板

這是一個專為屏東縣佳冬鄉「感潮型野生池」設計的極限作釣分析系統。本系統屏棄傳統「看農曆、憑感覺」的模糊經驗，改以**中央氣象署 (CWA) 實時數據**為基礎，透過底層物理機制（水流速度、溶氧梯度、Q10 代謝係數）進行建模，計算最精確的「魚口開啟」時窗。

---

## 🔬 核心科學架構 (First Principles Analysis)

本儀表板的預測邏輯基於以下科學底層機制，對每一項物質本質進行極限檢視：

* **趨流性機制 (Rheotaxis Trigger)**：
    針對感潮水域，系統精確鎖定「**漲七退三**」與「**漲三退七**」的關鍵物理時窗。此時潮差引發的流速劇增（10–30 cm/s）會觸發魚類的側線機械感受器 (Mechanoreceptor)，本系統對此進行 **1.5 小時** 的水位滯後補償校正。
* **Q10 溫度效應效應 (Metabolic Rate)**：
    利用攝氏溫度對魚類代謝速率的影響進行權衡。系統自動判別當前水溫是否處於目標物種（如鱸魚、花身鯻）的最適代謝範圍 ($T_{opt}$)，並據此調整活性百分比。
* **溶氧日週期梯度 (DO Daily Cycle)**：
    整合光週期數據，計算清晨（DO 谷底）與黃昏（光合作用 DO 巔峰）對不同耐氧力魚種的活性影響。
* **物理流體應對 (Gear Mechanics)**：
    根據潮汐引發的流體切應力，動態推薦釣組配置（如 PE 號數與咬鉛重），確保餌料在強流區仍能精確命中目標水層。

## 🛠️ 技術特點

* **實時資料對接**：透過 CWA API 獲取佳冬鄉當前天氣、風速、相對濕度及降雨機率。
* **動態魚群演算**：內建基於 HTML5 Canvas 的**生態模擬系統**，根據當日指數動態生成不同物種（如鱸魚、烏魚、土虱）的游動行為與密度。
* **黃金時窗時間軸**：自動掃描 24 小時內的「漲退流」區間，並以 A/S 級別進行戰術標註。
* **全物種百科**：收錄感潮水域常見 9 大物種的棲息水層、偏好餌料與捕食習性分析。
